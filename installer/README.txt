Thank you for choosing EasyRealtorsPro for your real estate management needs!

SYSTEM REQUIREMENTS
==================
- Windows 10 version 1903 (Build 18362) or later
- Windows 11 (all versions)
- Minimum 4GB RAM (8GB recommended)
- Minimum 1GB free disk space
- 64-bit operating system (x64 architecture)
- Internet connection for cloud features (optional)

INSTALLATION INSTRUCTIONS
=========================
1. Run EasyRealtorsPro_Setup_1.0.0_Production.exe
2. Follow the installation wizard prompts
3. Choose installation type:
   - Full: Includes all components, sample data, and documentation
   - Compact: Minimal installation for experienced users
   - Custom: Select specific components to install
4. Select installation directory (default: C:\Program Files\EasyRealtorsPro)
5. Choose additional options:
   - Create desktop shortcut
   - Create Quick Launch shortcut
   - Run at Windows startup
6. Click "Install" to begin installation
7. Wait for installation to complete
8. Launch EasyRealtorsPro when prompted

FIRST-TIME SETUP
================
1. Launch EasyRealtorsPro
2. Create your administrator account
3. Configure company information

TROUBLESHOOTING
===============
If you encounter issues during installation:

1. Ensure Windows is up to date
2. Temporarily disable antivirus software
3. Run installer as Administrator
4. Check available disk space
5. Verify system meets requirements

Common Issues:
- "Visual C++ Redistributable required": Installer will automatically install this
- "Insufficient disk space": Free up at least 1GB on the installation drive
- "Windows version not supported": Upgrade to Windows 10 version 1903 or later
- "Access denied": Run installer as Administrator

UNINSTALLATION
=============
To remove EasyRealtorsPro:
1. Go to Control Panel > Programs and Features
2. Select "EasyRealtorsPro"
3. Click "Uninstall"
4. Follow the uninstallation wizard

Note: Your data will be automatically backed up before uninstallation.
The backup location will be displayed during uninstallation.

DATA BACKUP
===========
EasyRealtorsPro includes automatic backup features:
- Daily automatic backups
- Manual backup creation
- Cloud backup support
- Data export capabilities

Backup locations:
- Local: C:\Users\[Username]\AppData\Local\EasyRealtorsPro\backup
- Cloud: Configured in Settings > Backup
